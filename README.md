# limbomass-app
Limbomass - Limitless web application for calculating BMI, BMR, and TDEE. This application does not collect, store, or share any personal data. All calculations are performed locally inside your browser.

The BMI, BMR, and TDEE results provided
by this application are estimates only.

They are not intended to replace
professional medical advice.

Always consult a healthcare professional
for medical diagnosis or treatment.
